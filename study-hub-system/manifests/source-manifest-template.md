### Classification Codes
- `LS` Lecture slides
- `LN` Lecture notes
- `TD` Tutorial document
- `AD` Assessment document
- `PR` Prescribed reading
- `CD` Case document
- `SE` Statutory extract
- `PP` Past paper
- `MG` Marking guidance
- `AI` AI-generated output (always Tier 3)
- `PN` Personal notes
- `UK` Unknown

### Tier Definitions
- `1` Primary — official university or publisher source
- `2` Supplementary — credible secondary source
- `3` Reference only — AI-generated, personal notes, unverified
- `X` Unusable — unreadable, corrupted, irrelevant

### Usage Status Options
- `Active` — currently used in the build
- `Superseded` — replaced by a newer version
- `Excluded` — consciously excluded (with reason)
- `Pending` — not yet processed
